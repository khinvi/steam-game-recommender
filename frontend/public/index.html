Steam Game Recommender System

A sophisticated machine learning application that analyzes user behavior and game metadata to provide personalized game recommendations. Built as an extension of a CSE258 project, using 7.8 million reviews from the Stanford SNAP dataset.

Technology Stack

Python FastAPI Scikit-learn Redis Docker NumPy Pandas

Key Features

✅ SVD with 26% Precision@K and 89% Hit Rate
✅ Multiple ML Models (Collaborative & Content-Based)
✅ Real-time Recommendations
✅ RESTful API with FastAPI
✅ Stanford SNAP Dataset (7.8M reviews, 51K+ games)
